# BookBase — iOS Book Discovery App

A SwiftUI iOS app for discovering and managing a personal book library.
Built as a Dicoding portfolio submission.

## Features
- Discover featured and recommended books
- Browse with genre filtering
- View detailed book information
- Save and manage favorites
- Search by title or author
- Offline mode with local JSON fallback

## Tech Stack
- Swift / SwiftUI (iOS 17+)
- Open Library API
- MVVM + @Observable
